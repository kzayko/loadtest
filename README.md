# Симуляция системы очередей

Программа для эмуляции нагрузки в системе с очередью и тремя этапами обработки.

## Описание системы

Система состоит из трех этапов обработки:
- **Этап 1**: Базовая обработка (20 сек)
- **Этап 2**: Три параллельные обработки (30 сек каждая, все три должны завершиться)
- **Этап 3**: Базовая обработка (12 сек)

## Параметры системы

- **Входящий поток**: 14 заявок/сек
- **Количество обработчиков**:
  - Этап 1: 2000
  - Этап 2: 1260 (каждая заявка требует 3 единицы ресурса)
  - Этап 3: 180
- **Время обработки**:
  - Этап 1: 20 сек
  - Этап 2: 30 сек
  - Этап 3: 12 сек

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python queue_simulation.py
```

## Результаты

Программа генерирует:
1. График количества заявок в очереди (поминутно)
2. График времени ожидания (поминутно)
3. График количества отказов (поминутно)

Графики сохраняются в файл `queue_statistics.png` и выводятся на экран.

## Настройка параметров

Все параметры можно изменить в начале файла `queue_simulation.py`:
- `INCOMING_RATE` - входящий поток заявок
- `STAGE1_WORKERS`, `STAGE2_WORKERS`, `STAGE3_WORKERS` - количество обработчиков
- `STAGE1_TIME`, `STAGE2_TIME`, `STAGE3_TIME` - время обработки
- `SIMULATION_TIME` - время симуляции (по умолчанию 1 час)
- `QUEUE_LIMIT` - лимит очереди для отказов
